# Задание 1

### Текст задачи

Решить задачу из лабораторной работы №2 (Задание 1. List.map) для
последовательности(На основе данного списка строк получить новый, где к каждой строке в конце
дописан заданный символ)

### Алгоритм решения

1)Запрашиваем у пользователя строку (через пробел)
2)Вводим строку
3)Разделяем строку по пробелам и получаем массив
4)Получаем последовательность из массива
5)Запрашиваем символ, который будем вставлять в конец каждой строки
6)Вводим символ
7)Вызываем  функция добавления символа в конец строки
7.1)seq.map принимает два параматера - функция преобразования (функция добавления символа в конец строки) которая применяется к каждой строке и исходная последовательность, в результате получаем новую измененную последовательность 
8)Выводим последовательность по строчке
